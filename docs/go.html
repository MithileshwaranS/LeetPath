API Documentation

This document outlines the endpoints and usage of the API. Each endpoint's details, including its purpose, required parameters, and sample requests, are provided.

Endpoints

POST /getSolvedQuestions

Retrieve solved questions for a user by their email.

Request Parameters:

  • email (string, required): The user's email address.

Sample Request:


fetch("http://localhost:8080/getSolvedQuestions", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ email: "user@example.com" })
})
.then(response => response.json())
.then(data => console.log(data));

Response:


{
    "status": true,
    "solved_questions": ["two-sum", "add-two-numbers"]
}

POST /addSolvedQuestion

Add a solved question for a user.

Request Parameters:

  • email (string, required): The user's email address.
  • question_slug (string, required): The question's slug.

Sample Request:


fetch("http://localhost:8080/addSolvedQuestion", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
        email: "user@example.com",
        question_slug: "two-sum"
    })
})
.then(response => response.json())
.then(data => console.log(data));

Response:


{
    "status": true,
    "message": "Question added to solved list"
}

POST /problemSet

Fetch a list of questions from LeetCode.

Request Parameters:

  • limit (query string, optional): The number of questions to fetch. Default is 50.

Sample Request:


fetch("http://localhost:8080/problemSet?limit=10", {
    method: "POST",
    headers: { "Content-Type": "application/json" }
})
.then(response => response.json())
.then(data => console.log(data));

Response:


{
    "total": 100,
    "questions": [
        {
            "title": "Two Sum",
            "difficulty": "Easy",
            "status": "ac",
            "topicTags": ["Array", "Hash Table"]
        },
        ...
    ]
}

POST /getUserData

Retrieve user data by email.

Request Parameters:

  • email (string, required): The user's email address.

Sample Request:


fetch("http://localhost:8080/getUserData", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ email: "user@example.com" })
})
.then(response => response.json())
.then(data => console.log(data));

Response:


{
    "_id": "64abc123ef456",
    "email": "user@example.com",
    "username": "user123",
    "solved_questions": ["two-sum"],
    "status": true
}

POST /addUser

Create a new user in the system.

Request Parameters:

  • email (string, required): The user's email address.
  • username (string, optional): The user's username.
  • institution (string, optional): The user's institution.

Sample Request:


fetch("http://localhost:8080/addUser", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
        email: "newuser@example.com",
        username: "newuser",
        institution: "Example University"
    })
})
.then(response => response.json())
.then(data => console.log(data));

Response:


{
    "status": true,
    "inserted_id": "64def123ab456"
}
